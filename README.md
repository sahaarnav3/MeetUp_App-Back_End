# ENDPOINTS  


| Purpose      | METHOD          | Endpoint  |
| ------------- |:-------------:| -----:|
|   Route to fetch HomePage.   | `GET` | / |
|   Route to a new event.   | `POST` | /events |
|   Route to fetch all events from the dataBase.   | `GET` | /events |
|   Route to fetch all event by ID from dataBase.   | `GET` | /events/:eventId |
